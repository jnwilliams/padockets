# PADockets Dataset

## Contents

This repository contains the dataset used in the paper, "A Bayesian Model of Cash Bail Analysis" (https://arxiv.org/abs/2101.12267). The zip file contains the dataset within a "padockets.csv" file, with a description and details on the contents under "datasheet.pdf".
